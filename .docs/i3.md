# i3 window manager

Manjaro i3 edition with Monokai bar colors.

## File

`~/.i3/config`

## Key bindings

| Binding | Action |
|---------|--------|
| `$mod+Return` | Open kitty |
| `$mod+d` | Application launcher (dmenu) |
| `$mod+j/k/l/;` | Focus left/down/up/right |
| `$mod+Shift+j/k/l/;` | Move window left/down/up/right |
| `$mod+1-8` | Switch workspace |
| `$mod+Ctrl+1-8` | Move window to workspace |
| `$mod+Shift+1-8` | Move and follow to workspace |
| `$mod+b` | Toggle back-and-forth workspace |
| `$mod+f` | Fullscreen |
| `$mod+h/v/q` | Split horizontal / vertical / toggle |
| `$mod+s/w/e` | Layout stacking / tabbed / toggle |
| `$mod+r` | Resize mode |
| `$mod+Shift+g` | Gap mode |
| `$mod+Shift+c` | Reload config |
| `$mod+Shift+r` | Restart i3 |
| `$mod+Shift+e` | Exit i3 |
| `$mod+m` | Toggle bar visibility |
| `$mod+0` | System menu (lock/exit/suspend/reboot/shutdown) |

## Window management

- **Border**: 1px pixel (`$mod+u` none, `$mod+y` pixel, `$mod+n` normal)
- **Gaps**: inner 14, outer -2
- **Smart gaps**: on (gaps only with >1 container)
- **Smart borders**: on (borders only with >1 container)
- **Focus follows mouse**: off
- **Back-and-forth**: workspaces toggle between last two
