# Monokai palette

| Token       | Hex       | Usage                       |
|-------------|-----------|-----------------------------|
| Background  | `#272822` | i3bar, kitty, tmux bar bg   |
| Foreground  | `#F8F8F2` | i3bar, kitty, tmux text fg  |
| Green       | `#A6E22E` | focused workspace, git +    |
| Orange      | `#FD971F` | binding mode, degraded      |
| Pink        | `#F92672` | urgent, error, bad          |
| Purple      | `#AE81FF` | accent                      |
| Blue        | `#66D9EF` | accent                      |
| Yellow      | `#E6DB74` | accent                      |
| Grey        | `#75715E` | separators, inactive items  |
| Dark Grey   | `#383830` | active workspace bg         |
| Selection   | `#49483E` | selection bg                |

## Color distribution

Monokai colors are mapped in kitty's 16-color ANSI table, the tmux status bar segments, i3bar workspace badges, and i3status status indicators — all using the same hex values for a consistent look across the desktop.
