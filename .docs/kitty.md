# Kitty terminal

## Config

`~/.config/kitty/kitty.conf`

## Font

```
font_family      Hack Nerd Font Mono
bold_font         Hack Nerd Font Mono Bold
italic_font       Hack Nerd Font Mono Italic
bold_italic_font  Hack Nerd Font Mono Bold Italic
font_size         11
```

## Features

- Tabs with Monokai-colored active tab
- Bell-inactive tabs dimmed
- Natural text input (Shift to override selection)
- Confirms on close if multiple kitty windows are open
- Syncs to paste (bracketed paste mode)
